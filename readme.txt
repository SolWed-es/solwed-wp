== Description ==

Solwed WP es un plugin profesional que integra múltiples funcionalidades para la gestión web:

* **Configuración SMTP avanzada** - Envío de emails profesional
* **Integración FacturaScript** - Conexión automática con CRM
* **Seguridad mejorada** - Control de intentos de login fallidos
* **Personalización de apariencia** - Banner personalizable
* **Editor de código** - CSS, JS y PHP personalizado
* **Sistema de logs** - Monitoreo de actividad

**Desarrollado por Solwed - Solutions Website Design**

== Installation ==

1. Sube la carpeta `solwed-wp` al directorio `/wp-content/plugins/`
2. Activa el plugin desde el panel de administración de WordPress
3. Ve a **Solwed WP** en el menú de administración
4. Configura los módulos según tus necesidades

== Frequently Asked Questions ==

= ¿Es compatible con PHP 8.1+? =
Sí, el plugin está optimizado para PHP 8.1 y versiones superiores.

= ¿Funciona con Elementor Pro? =
Sí, incluye integración automática con formularios de Elementor Pro.

= ¿Crea tablas nuevas en la base de datos? =
Solo las estrictamente necesarias para logs y seguridad, priorizando el uso de opciones de WordPress.
